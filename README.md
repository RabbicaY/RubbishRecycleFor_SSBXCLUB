# 生生不息生物社团垃圾分类闯关游戏项目程序

## 项目介绍

生生不息生物社团垃圾分类闯关游戏项目程序是一款基于Web的互动游戏，旨在通过游戏化的方式提高用户对垃圾分类知识的了解和掌握。游戏支持单人模式和多人对战模式，用户可以通过分类垃圾获得积分，并与其他玩家进行实时对战。

## 功能特性

### 核心功能
- **用户系统**：注册、登录、个人信息管理
- **游戏模式**：单人模式和多人对战模式
- **实时对战**：基于Socket.io的实时分数同步和结算
- **管理员系统**：管理匹配时间、监控对战房间、发布公告
- **排行榜**：记录用户积分排名
- **等待页面**：对战结束后等待对方上传分数

### 技术特性
- **前端**：HTML5、CSS3、JavaScript、Socket.io客户端
- **后端**：Node.js、Express、SQLite、Socket.io服务端
- **认证**：JWT token认证
- **响应式设计**：适配移动端和桌面端
- **实时通信**：基于Socket.io的实时数据传输

## 安装与运行

### 前置条件
- Node.js 18+ 环境
- npm 或 yarn 包管理器

### 安装步骤

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd 垃圾分类
   ```

2. **安装后端依赖**
   ```bash
   cd backend
   npm install
   ```

3. **配置环境变量**
   - 复制 `.env.example` 文件并重命名为 `.env`
   - 根据需要修改环境变量配置

4. **启动服务器**
   ```bash
   npm start
   ```

5. **访问游戏**
   - 打开浏览器，访问 `http://localhost:3000`

## 使用说明

### 普通用户
1. **注册账号**：在登录页面点击"用户注册"，填写姓名、学号和密码
2. **登录账号**：使用学号和密码登录
3. **选择游戏模式**：
   - 单人模式：直接开始游戏
   - 多人对战：进入匹配队列，等待其他玩家
4. **开始游戏**：根据提示将垃圾拖放到正确的垃圾桶
5. **查看结果**：游戏结束后查看得分，多人对战会进入等待页面等待对方完成
6. **返回主页面**：点击返回主页面按钮回到游戏首页

### 管理员
1. **登录管理员账号**：在登录页面点击"管理员登录"，使用管理员账号登录
   - 默认管理员账号：请自行阅读项目文档获取默认管理员账号
   - 默认管理员密码：请自行阅读项目文档获取默认密码
2. **管理匹配时间**：设置排位赛开放时间
3. **监控对战房间**：查看当前活跃的对战房间，可强制关闭
4. **发布公告**：发布公告，会在主页面顶部显示滚动文字

## 游戏规则

### 单人模式
- 游戏时长：20个垃圾物品
- 计分规则：
  - 正确分类：+1分
  - 错误分类：-0.5分
  - 连击3-5次：进入暴击模式，正确分类+2分
- 游戏结束后显示最终得分

### 多人对战
- 匹配方式：1v1 或 3v3
- 计分规则：与单人模式相同
- 结算规则：
  - 1v1：赢家获得100%积分，输家获得10%积分
  - 3v3：获胜队伍成员获得队伍总积分的50%，失败队伍成员获得队伍总积分的5%
  - 平局：双方各获得50%积分

## 常见问题

### 注册失败
- 确保学号未被注册
- 确保密码和确认密码一致
- 确保网络连接正常

### 匹配失败
- 确保处于匹配开放时间
- 确保有足够的玩家在线

### 无法返回主页面
- 尝试刷新页面
- 检查网络连接
- 点击等待页面中的"返回主页面"按钮
